* ERC721 Non-Fungible Token(NFT) 也就是非同质化代币
* 每个代币用一个数字来表示，例如:0、1、3、100。切确地说，应该是NFT合约地址 + 0、1、2数字来表示唯一NFT
* 每个代币都有自己的属性，这些属性同放在一个可以访问的.json文件。其中，文件里面包括文字、图片等...
* 与ERC20的区别在于：每个ERC20代币性质都是相同，它没有0、1、100数字来表示每一个代币,它也没有自己的属性
* [ERC721标准 https://eips.ethereum.org/EIPS/eip-721](https://eips.ethereum.org/EIPS/eip-721)
